## springboot-note

### springboot + 各种中间件和框架的笔记

主要目的:
* 记录 springboot 组合各种框架的简易使用方式
* 记录使用体验
* 方便抄代码

> 各组件及样例使用说明在 **对应的包/README.md** 文件中

目录结构

```
根目录
|----公共包
|----某框架名称
|----某框架名称
    |----README.md 笔记
    |----web
        |----controller
|----启动类
```
